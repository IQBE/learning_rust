# Learning Rust

In this repository, I will be learning Rust.

Rust is a programming language that is safe, fast, and efficient. I want to learn it because I believe it will be used more and more in the future. I think it will be a great language for me to learn as I have little to no experience with lower level languages.

Coming from a Typescript and Python background, I think that by learning Rust, I will be able to learn a lot more about how programs can be optimized and how to write better code. Eventually I would also make Rust my primary choice when it comes to programming backends.

## Structure

This repository is structured as a multi-project repository. It contains different folders that each contain a simple Rust project. The goal of each project is to demonstrate how to use Rust, not to create a single, complete application. Each project is a separate Rust project that should be ran independently.
